
## lv-blog

基于 `laravel` 的 `Blog` 应用。

lv-blog is based on the php framework laravel, you can use it to build blog easily!

使用方法：
- 执行 `git clone https://github.com/shaoyikai/lv-blog.git`
- 执行 `composer install`
- 修改根目录`.env`配置文件
- 执行 `php artisan migrate`
- 配置虚拟主机到`public/`目录下
